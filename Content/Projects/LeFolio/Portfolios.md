I built this portfolio and started experimenting with tools for building portfolios.
I started building a static website built with Vite and React, and deployed to GitHub Pages. This is the page you are looking at.
It's designed to be able to add projects with data while keeping the whole thing static so it can be hosted on GitHub Pages.

The first version was using yml, then I build another version for my psychology student portfolio, using markdown as source data.
It can be found at https://olivier-psy.fr

I ended up building a portfolio hosting platform with built in editor.
I abandoned the project but it is still live and you can create a portfolio with it!

Check out the live portfolio platform prototype at https://lefolio.fr.
