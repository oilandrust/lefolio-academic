AI coding agents have considerably lowered the bar for building small products, tools, and plugins that make life a little better. This plugin is an example.

Publishing Obsidian notes as a website is a problem that has established solutions. Obsidian offers a publishing service, Obsidian Publish, but it costs 8$ a month. Alternatively, we can use free tools such as a static site generator like Quartz and hosting on GitHub Pages to achieve a similar result for free. The downside is that we have to understand git, GitHub Pages, configuration, build, etc.

I used the power of AI to bridge that gap and implement a user friendly solution that allows to publish in one click leveraging existing free solutions. The principle is simple: we can automate all of the configuration and technical steps with a plugin and focus on making the experience smooth. While the idea is quite simple, coding this by hand is nontrivial and would take a talented developer a few weeks. With AI, I was able to analyze the technical steps, write specifications, validate the solution, and build a working solution in a few prompts and a few days.

The value here is critical thinking, we can let AI figure out the detail and focus on design and thinking about what brings value.
